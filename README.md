# Приложение "Журнал"


### Что это?
Это API - для сайта детского сада.


### Технологии
Python 3.5


### Установка зависимостей
```sh
$ source .virtualenvs/bin/activate
$ cd /path/to/doulog
$ pip install -r requirements.txt
```

### Запуск тестов
```sh
$ source .virtualenvs/bin/activate
$ cd /path/to/doulog
$ ./manage.py test
```
